1.这是什么
1.1 这是一个拉直曲线的插件
    本插件用于拉直Blender的曲线（Curve），拉直之后的各小段长度与原曲线小段长度一致。
    插件可以用于Curve实际包含多条曲线的情况。
    新直线的起点在原曲线（转换为mesh）后的第一个顶点上。

1.2 这是一个很好的例子，教你如何让你的插件具有多国语言界面
    直接改插件源程序里面的字符串的方法，比较傻。所以本插件展示了如何用gettext。
    本插件，默认提供了英语、简体中文界面。
    但是，我还不知道如何让bl_info支持gettext。不过用户应当阅读本文档获取帮助，
    而不是指望bl_info能给太多信息。如果哪位知道，请不吝赐教。

2.用法
2.1 基本用法
    在Blender的插件里面，搜索Straighten Curve，启用它。
    在3D窗口中，选择一个曲线（Curve）对象，在“工具栏”的Curve下，点击按钮，自动生成拉直的曲线。
    新生成的直线，继承原曲线的填充、深度、分辨率等参数。
2.2 切换语言
    在Blender的 用户选项-系统-语言 中选择需要显示的语言，左下角”保存用户设置“，下次重启后，
    再开启本插件，就会使用选定的语言（目前只支持简体中文、英文，如果选择了其他不存在的语言文件，
    自动使用英文）。
